# searchGifs

Приложение для показа и поиска гифок)  
Используется Kotlin, Coroutines, OkHttp, Glide, api сайта https://giphy.com/.  
Дополнительно реализовано:  
  - Пагинация  
  - Жест Swipe-to-Refresh
